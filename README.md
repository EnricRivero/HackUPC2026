# Calculadora Royale

Calculadora de matematicas simple hecha con Next.js, TypeScript y Tailwind CSS. La interfaz usa una estetica de arena fantastica inspirada en juegos tipo Clash Royale: azul rey, oro, cartas, escudos y botones grandes.

## Funciones

- Suma, resta, multiplicacion y division.
- Operaciones rapidas: porcentaje, raiz cuadrada y cuadrado.
- Boton de borrar, limpiar y cambio de signo.
- Historial corto de operaciones.
- Soporte para teclado: numeros, `+`, `-`, `*`, `/`, `Enter`, `Backspace` y `Escape`.

## Como correrlo en Visual Studio o Visual Studio Code

1. Abre la carpeta del proyecto en Visual Studio o Visual Studio Code.
2. Instala dependencias:
   ```bash
   npm install
   ```
3. Arranca la app:
   ```bash
   npm run dev
   ```
4. Abre la URL que muestra la terminal, normalmente:
   ```text
   http://localhost:3000
   ```

## Scripts disponibles

```bash
npm run dev        # Servidor de desarrollo
npm run build      # Compila la aplicacion
npm run start      # Ejecuta la version compilada
npm run typecheck  # Revisa tipos de TypeScript
```
